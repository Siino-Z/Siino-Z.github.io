💳 Credit Card Fraud Detection

Project Type: Machine Learning · Classification · Fraud Analytics

This project focuses on detecting fraudulent credit card transactions using machine learning techniques. A real-world-style dataset was used to train and evaluate multiple models including Logistic Regression, Random Forest, and XGBoost. The best-performing model achieved over 92% accuracy with a ROC-AUC of 0.976 after tuning.

🎯 Objectives

  • Train ML models to detect fraudulent transactions
  • Visualize transaction and fraud trends
  • Analyze model performance with evaluation metrics
  • Identify key features contributing to fraud
  • Save and prepare models for deployment

📊 Data Overview

The dataset included fields such as Transaction Type, Amount, Location, Time, Merchant ID, and a fraud label (IsFraud).

  • 🕐 Timestamps extracted into Hour, Day, and Month
  • 🗺️ Multiple cities across the U.S. represented in Location
  • 💳 About 1% of transactions were fraudulent

📈 Model Performance

Multiple models were trained and tested. The table below summarizes their final scores:

Model Accuracy ROC-AUC F1-Score
Logistic Regression 64% 0.66 0.64
Random Forest 97% 0.997 0.97
Tuned XGBoost 92% 0.976 0.92

🔍 Feature Importance

  • 🧾 Transaction Type and Amount were top indicators of fraud
  • 📍 Location and Merchant ID also contributed significantly

💾 Deployment Ready

The Random Forest model was saved using joblib and can be deployed via Flask or Streamlit for real-time prediction interfaces.
