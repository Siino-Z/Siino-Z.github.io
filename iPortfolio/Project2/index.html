Movie Recommendation Engine

Project Overview

This project showcases my proficiency in developing a machine learning recommendation system to predict user ratings for movies based on their historical interactions. The primary objective was to enhance personalized movie recommendations for users, similar to systems used by streaming giants like Netflix and Amazon Prime.

  • Dataset Used: MovieLens Dataset
  • Challenge: Predict how users will rate movies they haven't seen using collaborative and content-based filtering methods.

Technical Workflow

1. Data Wrangling and Exploration

Skills Demonstrated: Data Preprocessing, Feature Selection, Exploratory Data Analysis (EDA)

The initial phase involved handling large-scale datasets, merging information across multiple sources, and cleaning up irregularities like missing data. Key transformations included cleaning and normalizing user ratings, and extracting relevant metadata like movie genres, release dates, and actors.

Key Insights: Genre-based preferences, skewed ratings distributions.

2. Feature Engineering

Skills Demonstrated: Feature Engineering, Data Augmentation

I engineered features like average ratings per genre and user biases, helping the model align recommendations with user preferences. Temporal trends were also incorporated to enhance predictions.

3. Model Development

Skills Demonstrated: Collaborative Filtering, Content-Based Filtering, Hybrid Modeling

  • Collaborative Filtering: Used kNN, SVD, and SVD++ to recommend movies based on user-item interactions.
  • Content-Based Filtering: Built recommendations using movie metadata such as genres and cast.
  • Hybrid Model: Combined collaborative and content-based methods for more robust predictions.

4. Evaluation and Results

Skills Demonstrated: Model Evaluation, Hyperparameter Tuning, Visualization

Evaluated the models using RMSE to quantify prediction accuracy. The best-performing algorithm was Co-Clustering, with an RMSE of *1.13*.

The models were evaluated using RMSE, with Co-Clustering performing best in minimizing prediction error, followed by SVD++.

Key Insights & Skills Learned

  • Data manipulation and preprocessing for large datasets
  • Collaborative filtering, content-based filtering, and matrix factorization methods
  • Feature engineering, including temporal trends and user biases
  • Model evaluation using RMSE and hyperparameter tuning
  • Visualization with Seaborn and Matplotlib for deep data analysis

Conclusion & Next Steps

The Movie Recommendation Engine successfully predicted user ratings with high accuracy. Co-Clustering emerged as the best-performing model, reducing RMSE significantly. Moving forward, the next steps include hyperparameter tuning and experimenting with deep learning models like neural collaborative filtering to further enhance performance.
